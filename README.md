A professional, responsive email template for Zillow Listing Services policy updates.

## Features
- **Responsive Design**: Mobile-friendly layout using table-based structures.
- **Hosted Assets**: All images are hosted on GitHub to ensure they load reliably in email clients.
- **Direct Integration**: Pre-configured with specific Stripe subscription links.
- **Professional Branding**: Uses official Zillow horizontal logo and brand colors.

## File Structure
- `index.html`: Monthly subscription template.
- `yearly.html`: Yearly subscription template.
- `images/`: High-resolution assets (Logo, Icons).

## How to Use
1. Copy the HTML content from the desired file (`index.html` or `yearly.html`).
2. Paste it into your email sending service (e.g., SendGrid, Mailchimp, or custom SMTP).
3. Ensure the links are correct for your specific campaign.
